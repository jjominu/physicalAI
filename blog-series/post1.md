# 1편. 쉬운 사이트로 시작하는 웹 스크래핑

웹 스크래핑을 처음 연습할 때는 실제 쇼핑몰이나 로그인 페이지보다, 연습을 위해 공개된 사이트를 사용하는 편이 좋다. 구조가 단순하고 차단 위험이 적어 HTML을 읽고 데이터를 추출하는 기본 흐름에 집중할 수 있기 때문이다.

이번 예제에서는 `Books to Scrape` 사이트를 사용한다. 이 사이트는 책 제목, 가격, 재고 상태, 이미지, 상세 페이지 링크가 반복되는 카드 형태로 구성되어 있어 상품 목록 스크래핑을 연습하기 좋다.

## 목표

이번 글에서 만들 결과물은 간단하다.

- 책 목록 페이지에 접속한다.
- 각 책의 제목, 가격, 재고 상태, 이미지 주소, 상세 링크를 가져온다.
- 가져온 데이터를 Flask 웹 화면에 카드 형태로 출력한다.

## 기본 흐름

먼저 `requests`로 HTML 문서를 가져온다. 그다음 `BeautifulSoup`으로 HTML을 파싱하고, 책 하나를 나타내는 태그들을 반복해서 찾는다. 각 카드 안에서 제목, 가격, 재고, 이미지, 링크를 뽑아 리스트에 담으면 화면에 출력할 데이터가 준비된다.

```python
response = requests.get(url, timeout=10)
soup = BeautifulSoup(response.text, "html.parser")
items = soup.select(".product_pod")
```

이렇게 가져온 `items`를 반복하면서 필요한 값을 하나씩 꺼낸다. 스크래핑 코드는 사이트 구조에 영향을 많이 받기 때문에, 선택자를 정할 때는 브라우저 개발자 도구로 실제 HTML 구조를 확인하는 습관이 중요하다.

## 화면 출력

데이터를 가져오는 것에서 끝내지 않고 Flask로 화면까지 만들면 결과를 바로 확인하기 쉽다. `render_template_string`이나 별도 템플릿을 사용해 책 정보를 카드 형태로 출력하면, 스크래핑 결과가 제대로 들어왔는지 한눈에 볼 수 있다.

## 주의할 점

웹 스크래핑은 대상 사이트의 이용약관과 `robots.txt`를 확인하면서 조심스럽게 해야 한다. 특히 실제 서비스에 과도한 요청을 보내거나, 로그인 뒤에만 볼 수 있는 데이터를 무단으로 가져오는 방식은 피해야 한다. 학습 단계에서는 연습용 사이트를 사용하는 것이 가장 안전하다.
