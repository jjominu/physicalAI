# 데이터 수집부터 시각화까지 블로그 시리즈

## 1편. 쉬운 사이트로 시작하는 웹 스크래핑

초보자가 실습하기 좋은 공개 예제 사이트를 고르고, 상품명, 가격, 재고 상태 같은 기본 데이터를 가져오는 과정을 다룬다. HTML 구조를 확인하고 `requests`, `BeautifulSoup`으로 데이터를 추출한 뒤 Flask 화면에 보여주는 흐름까지 정리한다.

## 2편. CSV 데이터로 추이 그래프 만들기

수집하거나 직접 준비한 CSV 파일을 읽어서 날짜별 값의 변화를 그래프로 표현한다. `date`, `value` 컬럼 구조를 기준으로 데이터를 파싱하고, SVG 라인 차트로 결과물을 저장하는 방법을 설명한다.

## 3편. 작은 데이터 프로젝트를 GitHub에 정리하기

스크래핑 코드, CSV 시각화 코드, 문서 파일을 하나의 저장소 구조로 묶는 방법을 다룬다. README 작성, 폴더 정리, 커밋 메시지 작성, GitHub에 올릴 때 확인할 점을 정리한다.
